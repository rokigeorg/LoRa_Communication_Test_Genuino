#  LoRa Communication Ganuino Code

This repository includes the code for the Genuino to test the LoRa Communication between the node and the gateway. 

### Hardware Wiring - Node

#### RFM95 LoRa Modul - Arduino Uno
* 3.3V - 3.3V
* GND - GND 
* MISO - Digital 12  
* MOSI - Digital 11  
* SCK - Digital 13  
* NSS - Digital 10  
* DIO0 - Digital 3 
* RST - Digital 9
